 # React Component Library

Це Storybook-базована бібліотека UI-компонентів для React.

## Запуск
Встановити залежності:bash
npm install
npm run storybook
Відкрити у браузері:
http://localhost:6006

# Компоненти

## 1. Input

Типи: text, password, number

- Можливість очищення (clearable)  
- Для пароля: іконка toggle видимості
