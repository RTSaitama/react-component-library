 # React Component Library

Це Storybook-базована бібліотека UI-компонентів для React.

##  Запуск

1. Встановити залежності:
```bash
npm install

2. Запустити Storybook:

npm run storybook

3. Відкрити у браузері:

http://localhost:6006

 - **Компоненти:**

Типи
